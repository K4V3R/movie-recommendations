# Movie Recommendations

Базовый фронтенд-проект для поиска фильмов и сериалов через TMDb API.

## Что уже сделано

- Создан UI:
  - поле поиска
  - кнопка поиска
  - блок результатов
- Добавлены карточки фильмов (`.movie-card`) в минималистичном стиле.
- Подключен поиск через TMDb (`search/multi`).
- Реализован рендер результатов:
  - название (`title` или `name`)
  - рейтинг (`vote_average`)
  - описание (`overview`)
  - постер (`poster_path`) с fallback, если постера нет.

## Структура проекта

- `movies.html` — разметка страницы
- `movies.css` — стили интерфейса
- `movies.js` — логика поиска и рендера

## Запуск

1. Откройте `movies.html` в браузере  
   или запустите через локальный сервер (например, Live Server).
2. Введите название фильма/сериала.
3. Нажмите `Найти` (или Enter).

## TMDb API

В `movies.js` используется переменная:

```js
const API_KEY = "..."
```

При необходимости замените ключ на свой.

## Ограничения текущей версии

- Только клиентская часть (HTML/CSS/JS).
- Без бэкенда и без сохранения истории запросов.
- Без расширенной фильтрации (жанры, год, тип контента и т.д.).
